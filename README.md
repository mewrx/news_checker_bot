# News Checker Bot
> Telegram bot for checking news.

<img src="https://0x0.st/oows.png" alt="Earth" height="300">

# Requirements
- Python 3.8+

# Usage
Set your data into `setenv.md` and run it.

```bash
. ./setenv.sh
```

Run bot:

```bash
./bot.py
```
